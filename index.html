Mostly Serious Study Server

Welcome to the Mostly Serious Study Server webpage! The purpose of this webpage is to complement the Mostly Serious Study Discord Server and to offer a list of existing study materials, and additional useful resources. If you have any suggestions to improve this page, feel free to put them in the suggestions channel of the server!

About The Server

Server History

The Mostly Serious Study Server was intially created at the start of the 2020-21 school year by a couple of friends who wanted to share study material with each other, and to help each other with schoolwork & academics. Over time, the server has grown to include several dozen members with a similar goal in mind, and has amassed a fairly wide catalog of member-created & external study guides to attest to this. We hope that the server will continue to grow and help more people study & succeed. The Mostly Serious Study Server currently serves the Millburn Class of 2025.

Purpose of the Server

The purpose of The Mostly Serious Study Server is to provide a convenient platform for members to help each other with school & homework. Members can share study material, arrange study sessions, and ask questions regarding recent assignments or upcoming assessments. If you have created or have access to any study material that you think will be helpful for other people, we encourage you to share it with everyone in the server. In addition, if you have a question, whether it be related to class or the server itself, don't be afraid to ask.
